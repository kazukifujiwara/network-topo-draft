# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately via GitHub's
[private vulnerability reporting](https://github.com/kazukifujiwara/network-topo-draft/security/advisories/new)
for this repository — do not open a public issue for security problems.
You should receive an initial response within a week.

## Scope

The VSCode extension (`topodraft`), the CLI (`topodraft-cli`), and the
published JSON Schema. The extension loads no remote code and collects no
telemetry; the only bundled third-party library is jsonc-parser (see the
"Third-party software" section of the README).

## Supported versions

The latest released version only.
